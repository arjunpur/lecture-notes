This directory contains the lecture notes for \emph{Models of Higher Brain Function}. It uses the same book-style LaTeX structure as Models of Neural Systems, but the underlying template has been extracted into a reusable course-notes stack so the notes can grow independently.
The opening scaffold is intentionally compact: one overview chapter, a clean bibliography, and a minimal foundation for later material on cognition, memory, decision making, and cortical computation.
