This directory contains the canonical repository-backed lecture notes for \emph{Machine Intelligence II}. The scaffold uses the same durable notes structure as the richer \emph{Acquisition and Analysis of Neural Data} course repository: frontmatter, chapter files, backmatter, and a stable chapter map that the Bodhi lecture-notes service can target directly.
The opening chapter now focuses on the principal-component-analysis lecture. Future lectures can expand this repository chapter-by-chapter without reverting to ad hoc PDFs or one-off runtime overlays.
