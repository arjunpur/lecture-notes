This directory contains the lecture notes for \emph{Acquisition and Analysis of Neural Data}. The LaTeX structure was extracted from the Models of Neural Systems project and generalized into a reusable course-notes stack so the notes can grow independently of any homework or coding workspace.
The intent is to keep mathematical exposition, literature references, and course structure in one place. Lecture notes can expand chapter-by-chapter without coupling them to homework-specific code or submission artifacts.
The initial scaffold focuses on spike-train thinking: how neural signals are acquired, represented, cleaned, and transformed into interpretable statistical objects. The structure is deliberately minimal so future lectures can slot in without re-organizing the repository.
